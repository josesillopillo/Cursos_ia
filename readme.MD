# 🤖 DevOps Cursos IA

Proyecto final inspirado en la arquitectura **CIDE DevOps**. 
Este proyecto utiliza:
- Vanilla JS/HTML/CSS (Emulando estructura PWA/Angular en `src/` y `public/`)
- Contenedores Docker (Orquestados con `docker-compose.yml`)
- Infraestructura como Código (Terraform)
- K8s & GitOps (Manifiestos en `/argocd`)

## Instalación y Arranque

1. Clonar el repositorio.
2. Ejecutar `docker compose up -d --build`.
3. Navegar a `http://localhost:8081`.
